# Changelog

## v0.4.0 — 2026-07-11

- 项目由 Apple Value Lab 更名为 Value Lab。
- 重构为成本、市场、购买参考、生活平替、我的五个一级功能。
- iPhone 与 MacBook Pro 价值趋势从 1–4 年扩展到 1–5 年。
- 趋势图和表格新增逐年下降金额、环比下降率、累计下降金额、累计下降率和残值率。
- 增加“产品机龄”和“我的持有”两种趋势口径。
- iPhone 容量选择改为跟随机型联动，并支持缺失容量参考换算。
- 新增 108 条生活平替、18 个分类及可靠性/风险/证据评估。
- 新增生活平替搜索、筛选、详情、收藏和移动端卡片列表。
- 新增全局搜索。
- 新增“我的物品”和本地估值。
- 新增 JSON、市场价格 CSV、生活平替 CSV 导出。
- 完成深色主题、状态保存、移动端底部导航和错误状态。
