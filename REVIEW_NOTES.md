# 产品化检查与修改记录

本轮修改依据《网页开发中 AI 容易忽略的注意事项清单》，重点处理长期使用体验，而不是继续堆叠页面说明。

## 已修正

### 公开界面

- 删除公开页面中的 GitHub 文件路径、静态项目、后台实现等维护信息。
- 将“模型估算”改为面向用户的“参考区间”。
- 缩短标题和说明，保留必要的价格口径。
- 导出改为图标菜单，常用操作不再占用大面积文字按钮。

### 成本计算

- 修正原版本把“风险代理值”显示成金额的问题。
- 路径风险现在显示为低/中/高，不再冒充真实支出。
- 推荐路径会排除明显价格倒挂，再综合考虑成本和数据质量。
- 负成本或异常倒挂会显示“需核验”，不再被静默截断为零。

### 桌面与移动端

- 桌面端继续使用紧凑表格和右侧机型详情栏。
- 移动端改为固定底部导航，而不是压缩桌面导航。
- 移动端机型详情使用弹层，支持 ESC 和点击背景关闭。
- 表格保留横向滚动，不强行压缩复杂列。
- 修复 `backdrop-filter` 导致移动底部导航被限制在顶部栏中的定位问题。

### 主题与组件

- 新增完整深色主题，包括页面、表格、输入框、菜单、弹层、标签、边框和滚动条。
- 首屏会在 CSS 加载前恢复主题，避免明显闪白。
- 导出菜单支持点击外部关闭、ESC、上下键选择。
- 图标按钮均有 aria-label 和 tooltip。

### 状态与反馈

- 保存当前页面、成本参数、搜索筛选、主题和机型详情选择。
- 新增恢复默认参数。
- JSON/CSV 导出完成后显示轻量 toast。
- 增加无结果状态、数据加载失败状态和重新加载入口。

### 安全与健壮性

- 动态文本统一转义。
- 外部链接仅允许 HTTP/HTTPS，并使用 `noopener noreferrer`。
- 导出文件名使用数据日期，不再写死版本日期。
- JSON 导出不包含临时界面对象，只包含可维护数据。
