## 全站逻辑原则

```text
选择对象
→ 选择配置或条件
→ 展示事实价格与参数
→ 计算结果
→ 比较路径
→ 给出建议
```

普通进入页面时，不自动替用户选择产品列表或平替列表中的第一项。只有用户点击、全局搜索跳转或其他页面明确带入时，才展示具体详情。
