# Value Lab 开发计划

## 阶段 1：通用框架与生活平替 MVP — v0.4.x

### 已完成

- 五个一级功能。
- iPhone / MacBook Pro 1–5 年价值趋势。
- 逐年下降金额和百分比。
- 生活平替搜索、分类、筛选、详情和收藏。
- 我的物品和浏览器本地保存。
- 移动端独立列表和底部导航。

### 下一步

- 给生活平替增加真实价格样本和来源链接。
- 增加用户纠错数据导出格式。
- 增加平替条目编辑工具，供维护者生成 JSON。
- 将公开平替从 108 条扩充到 300 条以上，但只发布审核通过的条目。

## 阶段 2：Apple 产品补全 — v0.5.x

- iPad 标准版、mini、Air、Pro。
- MacBook Air。
- iPhone 8、X、XS、XR 等历史机型。
- 每个 iPhone 容量的独立首发价格和市场价格，不再主要依赖容量换算。
- MacBook 芯片、内存和 SSD 多级联动。
- 配置升级投入、二手溢价和不可回收配置成本。

## 阶段 3：Android 与 Windows 电脑 — v0.6.x

- 华为、荣耀、小米、OPPO、一加、vivo、iQOO、三星。
- ThinkPad、联想、戴尔、惠普、华硕、Surface。
- 系统支持年限、安全更新年限和售后维修价格。
- Android 发布后 3 个月、618、双 11 和一年二手价格节点。

## 阶段 4：影像与存储 — v0.7.x

- GoPro、DJI Action、Insta360。
- DJI Mini、Air、Mavic、Avata。
- 相机机身与镜头。
- NAS、机械硬盘和 SSD。
- 飞行时间、电池循环、炸机和维修记录。
- 硬盘通电时间、SMART、TBW 和健康度。

## 阶段 5：自行车与组合资产 — v0.8.x

- 捷安特、美利达、喜德盛、闪电、崔克等。
- 车架、套件、轮组、功率计和升级件。
- 整车出售、恢复原装出售和拆卖比较。
- 改装投入、升级件残值和每公里成本。

## 阶段 6：汽车 — v0.9.x

- 独立 10 年价值模型。
- 里程、事故、过户、保险、购置税、融资、保养、轮胎和能源。
- 新能源电池健康、快慢充次数、智驾硬件和厂商降价。
- 每月和每公里总持有成本。

## 阶段 7：箱包、手表与可能升值物品 — v1.0

- 价值趋势允许上涨，不再只显示下降。
- 真伪鉴定、寄卖佣金、流通速度和波动率。
- 保卡、附件、维修、抛光和热门款因素。

## 数据维护工具

每个阶段都同步建设：

- CSV / JSON 批量导入。
- 重复检测和别名归一。
- 高风险关键词扫描。
- 来源、样本数和更新时间。
- 数据变更记录。
- 自动生成 Changelog。
