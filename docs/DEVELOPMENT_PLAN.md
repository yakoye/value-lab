# Value Lab 开发计划

## 阶段 1：通用框架与生活平替 MVP — v0.4.x

### 已完成

- 五个一级功能。
- 生活平替搜索、分类、筛选、详情和收藏。
- 我的物品和浏览器本地保存。
- 移动端独立布局。

### 后续维护

- 给平替增加真实价格样本和来源。
- 建立维护者条目编辑、去重和审核工具。
- 将公开平替逐步扩充到 300 条以上。

## 阶段 2：Apple 产品补全 — v0.5.x

### v0.5.0 已完成

- iPad 标准版、mini、Air、Pro，共 18 款。
- MacBook Air M1–M5，共 9 款代表配置。
- iPhone 8、8 Plus、X、XR、XS、XS Max。
- 四类产品共用成本、市场、购买参考、搜索和“我的物品”。
- iPad 容量与 Wi-Fi / 蜂窝网络联动。
- 官方新机容量价差与二手容量残值分开计算。

### v0.5.x 下一步

- 为 iPhone / iPad 每个容量补充独立的真实市场样本。
- MacBook Air / Pro 增加内存与 SSD 多级联动，而不只是代表配置。
- 增加配置升级投入、当前二手溢价和不可回收配置成本。
- iPad 增加 Apple Pencil 与键盘配件成本、残值和套装出售比较。
- 增加价格数据维护页和 CSV 批量导入校验。

## 阶段 3：Android 与 Windows 电脑 — v0.6.x

- 华为、荣耀、小米、OPPO、一加、vivo、iQOO、三星。
- ThinkPad、联想、戴尔、惠普、华硕、Surface。
- 系统支持年限、安全更新年限和售后维修价格。
- Android 发布后 3 个月、618、双 11 和一年二手价格节点。
- Windows 笔记本内存、SSD、显卡、屏幕和保修配置联动。

## 阶段 4：影像与存储 — v0.7.x

- GoPro、DJI Action、Insta360。
- DJI Mini、Air、Mavic、Avata。
- 相机机身与镜头。
- NAS、机械硬盘和 SSD。
- 飞行时间、电池循环、炸机和维修记录。
- 硬盘通电时间、SMART、TBW 和健康度。

## 阶段 5：自行车与组合资产 — v0.8.x

- 捷安特、美利达、喜德盛、闪电、崔克等。
- 车架、套件、轮组、功率计和升级件。
- 整车出售、恢复原装出售和拆卖比较。
- 改装投入、升级件残值和每公里成本。

## 阶段 6：汽车 — v0.9.x

- 独立 10 年价值模型。
- 里程、事故、过户、保险、购置税、融资、保养、轮胎和能源。
- 新能源电池健康、快慢充次数、智驾硬件和厂商降价。
- 每月和每公里总持有成本。

## 阶段 7：箱包、手表与可能升值物品 — v1.0

- 价值趋势允许上涨。
- 真伪鉴定、寄卖佣金、流通速度和波动率。
- 保卡、附件、维修、抛光和热门款因素。

## 每个阶段同步建设

- CSV / JSON 批量导入。
- 重复检测和别名归一。
- 来源、样本数、更新时间和可信度。
- 数据变更记录和自动生成 Changelog。


## v0.6.0 已完成

- Android 手机首批代表机型
- ThinkPad 首批代表机型
- GoPro / DJI / Insta360 运动影像
- DJI 无人机
- NAS 与硬盘/SSD
- 自行车首批品牌与整车

## 下一阶段

- 扩充每个新类别的型号和真实市场样本
- 增加健康度、飞行次数、通电小时、骑行里程等品类专属状态输入
- 加入汽车和高价值箱包/手表的独立价值模型

## v0.6.1 已完成：页面逻辑层级重构

- 成本页改为六步依赖顺序。
- 市场页新增机型、配置和日期筛选，价格只比较同一配置。
- 购买参考不再自动选中第一款产品。
- 生活平替不再自动选中第一条方案。
- “我的”空状态不再显示无意义的 0 元汇总。
- 添加物品时保存明确的容量、内存、SSD和网络版本。
- 补充现状思维导图、逻辑审查和重构计划文档。

## v0.6.x 下一步

1. 统一 Product / Configuration / Market Price 数据结构。
2. 为 Android、无人机、NAS、硬盘和自行车增加品类专属状态输入。
3. 建立真实价格历史与平台价格中心。
4. 扩展“我的物品”为资产变化、保养记录和出售建议。
5. 在完成数据模型统一前，不继续大规模增加新品类。
