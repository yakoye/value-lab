# 生活平替审核说明

首批数据从用户整理的生活平替清单中选取，完成了以下处理：

1. 统一“原需求 → 平替方案”格式。
2. 合并同义词和重复用途。
3. 将线材收纳等高度重复内容合并为一个标准条目。
4. 删除“某物 → 某物”式自我替代。
5. 删除只有“换成平价产品”但没有明确功能和限制的模糊条目。
6. 将功能可靠性、安全风险和证据等级拆开评价。
7. 不公开高风险健康、宠物用药、婴幼儿安全、危险化学品、电器改装、火源和关键承重方案。

## 首批数据

- 公开平替：108 条。
- 分类：18 个。
- 隔离的高风险示例：8 类。

## 公开条件

每条公开平替至少需要：

- 清晰的原需求和平替名称。
- 可解释的功能对应关系。
- 明确的限制。
- 明确的安全或材质条件。
- 可靠性、风险和证据等级。

## 不能作为可靠证据的内容

- 单纯点赞量。
- “某某同厂”或“某某代工厂”但无材料证明。
- 只有外观相似。
- 单一极低价商品链接。
- 未经验证的医疗、宠物营养或儿童安全建议。
